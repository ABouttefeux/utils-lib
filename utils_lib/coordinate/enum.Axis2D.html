Enum utils_lib::coordinate::Axis2D

source · 
pub enum Axis2D {
    Vertical,
    Horizontal,
}
Expand description

Represent the Axis in 2 dimensions. It can be either in the x direction i.e. Self::Vertical or the y direction, i.e. Self::Horizontal.
